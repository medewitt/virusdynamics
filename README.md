# virusdynamics

An attempt to reproduce the code from *Virus Dynamics: Mathematical Principles of Immunology and Virology* by Nowak and May.

No guarantees that anything in this is correct; it is just an attempt to reproduce what is in the book.

The book is built with [Quarto](https://quarto.org/docs/books) and rendered into the `docs/` directory. Run `make` (or `quarto render`) to rebuild it.
